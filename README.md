# SoloOS

一个基于 React + Vite 构建的现代化 Web 应用程序。

## 🌐 在线访问

**直接点击链接即可使用，无需安装：**

👉 [**https://dannychen98.github.io/danny98**](https://dannychen98.github.io/danny98)

---

## 🚀 快速开始

### 在线使用（推荐）
直接访问上面的链接，在浏览器中即可使用全部功能。

### 本地开发

```bash
# 克隆项目
git clone https://github.com/Dannychen98/danny98.git

# 进入项目目录
cd danny98

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

然后打开浏览器访问 http://localhost:5175

---

## 📦 技术栈

- **React 19** - 现代 React 框架
- **Vite** - 极速构建工具
- **Tauri** - 桌面应用框架（可选）

---

## 🛠️ 构建部署

```bash
# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
